# Imperativo afirmativo

<b> Der bejahte Imperativ</b>
<br />
<br /> Der Imperativ ist die Befehlsform. Diese kann sich sowohl an eine Person als auch an mehrere Personen wenden, z.B. Sei bitte leise! oder Seid bitte leise!
